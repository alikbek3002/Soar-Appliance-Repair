WHAT'S INSIDE

  SVG/  — VECTOR files. Scale to any size with NO quality loss.
          Use these for PRINTING: t-shirts, business cards, signage,
          banners, vinyl/embroidery, large format. Preferred by print shops.
            soar-logo-color.svg  — full color (navy + blue)
            soar-logo-black.svg  — single color, black  (1-color print / engraving)
            soar-logo-white.svg  — single color, white  (for dark backgrounds)

  PNG/  — Transparent-background images (2400 px wide). For web, email,
          social media, documents, presentations.
            soar-logo-color.png  — full color
            soar-logo-black.png  — black
            soar-logo-white.png  — white (use on dark backgrounds)

  PNG-on-background/ — ready-to-use on a solid background.
            soar-logo-on-white.png — color logo on a white card
            soar-logo-on-navy.png  — white logo on a navy card


WHICH ONE DO I USE?
  • Printing anything (shirt, card, sign)  ............  SVG (color, or black for 1-color)
  • Light / white background  ........................  color or black version
  • Dark / navy / photo background  ..................  WHITE version
  • Website / email / social  ........................  PNG (transparent)
  • Need an exact color match  .......................  give the printer the HEX/RGB above


Questions about formats (EPS, PDF, Pantone, specific sizes)? Just ask —
the logo is vector, so it can be exported to any format or size on request.
